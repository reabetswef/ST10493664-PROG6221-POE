# Cybersecurity Chatbot POE

A .NET 8-based cybersecurity chatbot designed to assist users with cybersecurity awareness, threat explanations, and general security guidance. This project demonstrates chatbot logic, secure coding practices, and modern .NET development.

## Features
- Interactive chatbot interface
- Cybersecurity-focused responses (phishing, malware, password safety, etc.)
- Expandable rule-based response system
- Built with .NET 8
- Testable and modular architecture
- Designed for future AI or API integration

## Tech Stack
- .NET 8 (C#)
- Console Application or ASP.NET Core (depending on implementation)
- GitHub Actions for CI/CD
- Optional AI/LLM integration (future enhancement)
